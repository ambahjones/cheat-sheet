HTML Elements and Attributes

HTML Elements

Tag Name Use
<head> Head Contains metadata and gives the browser information about the page itself as well as the title of the page.
<title> Page title Contains the tile of the page that will display at the top of the browser tab or window. Housed within the <head>.
<body> Body Only contents enclosed in this element will be displayed on the webpage.
<h1> - <h6> Headings Numbers (1-6) indicate importance and text size within heading. They are displayed in bold text.
<div> Division A container to divide the page into sections that is useful for grouping elements together and customizing CSS.
<p> Paragraph Use to insert a block of text into a webpage.
<span> Span Use to separate a short piece if text on the same line as other content for alternate styling
<em> Emphasize Used to emphasize text, is usually displayed in italics on webpages.
<strong> Strong Used to highlight text, usually displayed as bold text on webpages.
<br> Line break Aids in creating space in the visual representation of code on a website that adding a line in the code will not accomplish. Does not have a closing tag.
<ul> Unordered list Used to start an unordered list that will display with bullet points.
<ol> Ordered list Used to start an ordered list that will display with numbers before the list items.
<li> List item Must be added inside <ul> and <ol> tags to create the list.
<img> Image Self closing tag used to add an image to your page. Image tags require source and alt text attributes.
<video> Video Used to add video media to your page. Has a closing tag and requires source, width, and height attributes. Can also add a controls attribute.
<a> Anchor Used to create a link to another page, either part of your website or outside of it. Use the target attribute to open this link in a new tab or page.

Semanitc HTML Elements

These elements are used to make your site structure easier to understand, improves its search engine optimization (seo), and helps to make the site more accessible to mobile devices and users accessing with a screen reader.

Tag Name Use
<header> Header A container for navigation links, introductory content or headings (<h1>-<h6>).
<nav> Navigation Used to define a block of navigation links (menus, tables of contents, etc.) It can be used inside <header> but this is not strictly necessary.
<main> Main Contains the dominant content of the webpage. More descriptive than using <div>, allows screen readers to better determine where the main content of the page is located.
<footer> Footer Located at the bottom of a webpage the footer contains information such as contact information, copyright information, terms of use, site maps and reference and top of page links.
<section> Section Defines elements in a document with similar themes
<article> Article Holds standalone content like articles, blog posts, comments, etc.
<aside> Aside Used for additional information that enhances another element but isn't required to understand the main content such as bibliographies, end notes, comments, etc.
<figure> Figure Used to encapsulate media for when you want to add an image, illustration, diagram, code snippet, etc.
<figcaption> Figure caption Used to add a caption describing the content of a figure element. It is added inside <figure> and will move with the element if relocation is necessary (unlike using <p>).
<audio> Audio Used to embed audio content in a webpage using a source link to the file. The type of audio can be specified using the type attribute, helping the browser identify the audio content more easily. Control can also be used on this element.Has a closing tag.
<video> Video Used to define video media added to a page, utilizes attributes to alter video playback like controls, autoplay and loop. Has a closing tag.
<embed> Embed Embed can incorporated any type of media content on a page, meaning having an embed button on a site means it contains embedable media. It is a self closing tag and is a depreciated element but may still be present in legacy code.

HTML Attributes

Attributes are added to the opening tag of an element to expand the tag with additional information or change the styling of that element. They consist of a name and a value.

Attribute Name Use
class Class Second lowest in specificity for selector attributes, a class is meant to be used over multiple elements when you want to style the CSS of those elements in the same fashion.
id ID An id should only be given to one element as it is the most specific selector you can use. Id allows and element to be styled independantly of any other tag or class and will override any existing CSS on elements it is applied to if there are contradicting styles.
src Source The src attribute is used to indicate the origin of a link, image, or other media file that you want to embed in your site.
alt Alternative text The alt attribute allows you to add alternative text describing an image in case the image can't be displayed by the browser. It also gives screen readers access to the contents of the image for visually impared users and helps in search engine optimization (seo).
controls Controls The controls attribute adds a set of basic playback controls (play, volume, etc) for video and audio media.
href Hypertext reference Add href="" to <a> or <link> to indicate the destination for the link. It can be a URL or a relative path if the link is to a document on the same site.
target Target Target specifies where to open a linked document when clicking on <a> text. Setting its value to "_blank" will open the linked document in a new window or tab.
width Width Width can be used on <img> and <video> to specify the width of the media on the page.
height Height Height can be used on <img> and <video> to specify the height of the media on the page.
type Type Defines the type of the element. Used on <audio>, <button>, <input>, etc to give the browser more information about the element of type of media.
autoplay Autoplay Indicates audio or video should play upon loading.
loop Loop Indicates that the audio or video should play on a continuous loop.
